試験当日でした

学習時間を無駄に費やしてしまったというぐらいには簡単だった。両方共500点の合格ラインに対して700点超え。昨日学習をしなかったことがマイナスに響くのではないかと実は怯えていたのだけれど、そんなことはなかった。でも、直前でping-tの模擬試験やってみたらできてたはずの問題ができなくなっていたので怯えてはいた。

でも、特にLpic-101に関しては出た問題の運が良かっただけかもしれないけれど、学習する前の知識量でも合格できたのでは？という思いが強い。試験の詳細については喋っちゃダメだよというのを試験前に確認させられるので記載しないけれど…。でも日本語の内容と英語の内容が全く違う問題が出たから本当に焦ったぞ、どっちで答えれば良いのかもわからんし。テストのバグかもしれないからな。 　
　

本当は現在無職である

なんとなく恥ずかしかったので（とは言え、学習時間から推測できるが）ここでは記載していなかったが、現在無職であり、Lpicの取得は入社試験も兼ねていた。正直、自分はかねがねLpicは取得はしていないけれど知識はあるよと思っていたので馬鹿にしているのかしら？と思った（経歴を見ればそれぐらいの知識はあるであろうと思ったのだが）。しかし結局目に見える成果がなければダメなのだろうなあ。と考えな直して勉強した。

結果として、自分が理解していないところを埋められたし、資格も取れたのでプラスのことが多かった。でもXとかプリンタとかそんなもん使う機会はやっぱり無いと思う。systemdとかipコマンドとかいつかわからなきゃいけないなと思っていたものをきちんと学び直せたのが良かったところだと振り返って思う。

でもちょっと頑張りすぎ。合格できれば良いのだからはしゃがなければよかったよ。

明日はちょっとお休みすると思う。ちょっとだけ疲れた。映画を久しぶりに見に行くかもしれない。わからん。

それでも、間を開けないでLpic2は勉強しようかなあと思う。今は波に乗っているしね。でも資格取得にそこまで興味あるわけでもないんだよなあ。
